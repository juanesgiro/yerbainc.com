# Yerba Inc. | Premium Smoke Shop

Static website for Yerba Inc. — a locally owned smoke shop featuring premium glass, accessories, vaporizers, and more.

## Pages
- **Home** — Hero, product carousel, about, contact
- **Inventario** — Visual product catalog filtered by category
- **Galería** — Photo gallery of the store
- **Promociones** — Current deals and offers
- **Blog** — Articles and updates
- **Admin** — Product management panel (localStorage-based)
